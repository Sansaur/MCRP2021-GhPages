Short description

Being any kind of magic user immediately makes you a spellslinger as well, giving you access to tools all magic users need

Gains access to:

  • Wizard clothes
  • Warlock clothes
  • Battlemage Armor (If you also have Heavy Armors)
  • Gilded wood
  • Magical lecterns
  • Receptacles
  • Imbuement altars
  • Arcane workbench
  • Generic magic baubles
