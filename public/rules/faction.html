Faction creation

Factions need a minimum of 2 members and activity by at least the same amount of people

If a faction remains fully inactive their territories will be kept as "standby" for up to a month, afterwards everything inside will be deleted and they'll become "ruins"

Requirements:

  • Name of the faction
  • Leader's character
  • Coordinates of the main base (The whole biome will belong to the faction) + Which biome is it
  • Starting members
  • Backstory
  • Purpose
  • Member rules
  • Standing with other currently existing factions

It's possible to have an "organic" start instead, without as much focus on the faction's backstory and instead build it up as you go, you still have to fill all the fields given beforehand, but at the start of the server you can begin with just these:

  • Leader's character
  • Which biome do you want (I'll pick the coordinates)
  • Starting members
