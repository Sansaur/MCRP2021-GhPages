Character creation

To create a character

To create a character you only need to publicly post the following in #character-creation in Discord, and ping any admin (In case we add a whitelist)

  • Full name and ingame name (Ingame names tend to not be able to hold the full name)
  • Faction you'll be a part of (If you want to be leader of a new faction, create it first)
  • Pick 3 Classes your character will have OR talk to an admin to grant you special roleplay status
  • Pick if you're Restricted or Wild in PVP, and offer the whitelist of actions if Restricted
  • Pick a religion you want to be a follower of (If you want to be atheist go for it, and if you want a new one, create it first)
  • Offer a physical description of your character
  • Offer a backstory to your character that follows the events until now
  • Offer a picture of your character (It's better if it's from outside the game)
